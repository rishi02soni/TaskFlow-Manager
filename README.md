# TaskFlow Manager

TaskFlow Manager is a simple and efficient console-based task management application built using **.NET** and **C#**. It allows users to create, manage, complete, and delete tasks with automatic file saving support.

---

## 🚀 Features

- Add new tasks
- View all tasks
- Mark tasks as completed
- Delete tasks
- Auto save data to file
- Load saved tasks on startup
- Clean console UI
- Built with OOP concepts

---

## 🛠️ Technologies Used

- C#
- .NET Console Application
- File Handling
- Object-Oriented Programming
